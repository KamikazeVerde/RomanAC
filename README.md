This project is open-source and community-driven.

## Checks
- Scaffold:
    - F (Limit Exceeded)
    - B (Perfect Clicking)
    - D (Invalid Snaps) (!)
    - C (Keep Y) (!)
- Combat:
    - Aura A (Tickrule)
    - IrregularCombat (Heuristics)
- Movement:
    - Movement A (Groundspoof/Fake Ground)
    - Movement X (Analysis/Heuristics)
